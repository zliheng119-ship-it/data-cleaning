# 电池数据工具包使用说明

这是一个可移动的本地工具包。把整个 `电池数据工具包` 文件夹复制到其他项目后，仍然可以使用。

## 当前功能

```text
.cex/.nda/.ndax -> 电压随时间变化曲线
```

输出结果只包含：

```text
plot.svg
*_voltage_time_data.xlsx
```

## 第一次使用

先双击：

```text
安装依赖.bat
```

这会在工具包文件夹内部创建一个 `.venv` 环境，并安装：

```text
pandas
numpy
matplotlib
openpyxl
NewareNDA
```

## 绘制电压-时间曲线

最简单方式：

```text
把 .cex/.nda/.ndax 文件拖到 绘制电压时间曲线.bat 上
```

也可以在命令行运行：

```bat
python -m battery_data_tool voltage-time --input "D:\project\data.ndax" --output-dir "D:\project\result"
```

## 输出位置

如果没有手动指定输出目录，会在输入文件旁边生成：

```text
原文件名_voltage_time_output
```

里面包含：

```text
plot.svg
原文件名_voltage_time_data.xlsx
```

## 修改出图格式

默认模板在：

```text
templates\voltage_time_style.json
```

可以修改字体、字号、坐标轴宽度、刻度宽度、曲线线宽、图尺寸、坐标轴范围等。
