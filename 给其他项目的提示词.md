# 给其他项目大模型的提示词

把下面这段话复制到其他项目的大模型对话里，或放到那个项目的 `AGENTS.md` / `README.md` / `工具说明.md` 中。

```text
本项目可以调用一个本地电池数据处理工具包，路径是：当前项目中的“电池数据工具包”文件夹。

当我需要处理 .cex、.nda、.ndax 电池原始数据，并绘制“电压随时间变化曲线”时，请优先调用这个工具包，而不是重新写脚本。

调用方式：

1. 进入“电池数据工具包”目录。
2. 如果存在 .venv\Scripts\python.exe，优先使用它。
3. 运行：

   .venv\Scripts\python.exe -m battery_data_tool voltage-time --input "原始数据文件路径" --output-dir "输出文件夹路径" --time-unit h

   如果没有 .venv，则运行：

   python -m battery_data_tool voltage-time --input "原始数据文件路径" --output-dir "输出文件夹路径" --time-unit h

4. 输出结果应包含：
   - plot.svg
   - *_voltage_time_data.xlsx

5. 不要修改原始数据文件。输出目录应放在当前项目的数据结果文件夹中。

如果缺少依赖，请先在“电池数据工具包”目录运行“安装依赖.bat”。
```
